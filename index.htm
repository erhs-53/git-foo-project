The upper portion of this main page was created with a different color of text to highlight something specific you add to your site. You can change any aspect of this design, right down to the 'Site Name' text. You can do this by making adjustments to the .PSD file that is included with this template. The top portion of this template is one complete .PSD file that has been sliced in Photoshop© 6 and Image Ready© 3.0. If you need additional help with this template, you can contact us and will do our best to help you. You can also find additional help from sources like Quackit - of particular relevance is it's HTML codes section, and it's HTML Tutorial.

This template also uses a specially designed background file that is called bac1.gif that can be found in the 'images' folder included with your template. It is important that you upload this entire file to your web server when you are uploading your new template. It contains all the upper image slices, this spiffy background, and the background images used in the boxes to the left (Site News, Member Log In). Keep in mind, there are 4 different box header background files for you to use. You can find out in the HTML code which files are used in this template. You can change them if you like. These can also be found in the 'images folder'.
